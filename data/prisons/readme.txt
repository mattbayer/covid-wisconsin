Data in prisons-wi.csv culled from 

WI DOC weekly population report, 8-Jan-2021:
https://doc.wi.gov/DataResearch/WeeklyPopulationReports/01082021.pdf
WI DOC Covid dashboard, accessed 11-Jan-2021:
https://doc.wi.gov/Pages/COVID19%28Coronavirus%29/COVID19TestingDashboard.aspx

PDFs and Excel versions of these data are saved in this folder.
prisons-wi.csv collates both together into machine-readable csv format.

prisons-wi-covid.csv filters on only institutions with Covid data, excluding mainly contract
beds with county jails.
